# SLS handler factory

Some helpers for easily manage serverless handlers creation.

### Prerequisites

Requires aws-sdk package (if executed in AWS Lambda, it is already installed).

```
npm install --save-dev aws-sdk
```

## npm scripts

Build the js files from typescript:
```
npm run build
```

Running tests:
```
npm run test
```

Running style check:
```
npm run style
```
